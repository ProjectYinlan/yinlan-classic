洇岚 是一个基于 Mirai (mirai-ts) 服务于 Furry 文化群聊的 QQ Bot。

## 关于“经典版”

因为同时建立了 Yinlan-next 的计划，于是出现了经典版。

经典指与原洇岚同样使用 JavaScript ，而 next 使用 TypeScript。
